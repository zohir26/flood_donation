Important FAQ

What is the Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each object corresponds to a part of the document, such as elements, attributes, or text. The DOM provides a structured way to access and manipulate the content, structure, and styling of a document through scripting languages like JavaScript.

How do you select an element from the DOM?

In JavaScript, we can select elements from the DOM using various methods provided by the document object. These methods allow us to target elements based on different attributes like ID, class, tag name, or CSS selectors.
Common Methods to Select DOM Elements: 1. getElementById() 2. getElementsByClassName() 3. getElementsByTagName() 3.querySelector() 4. querySelectorAll() 5. getElementsByName() etc

What is an event delegation in the context of DOM and why it is useful?

Event delegation is a technique in JavaScript where we can add a single event listener to a parent element instead of adding multiple event listeners to individual child elements. The event listener on the parent can handle events triggered by its children due to event bubblingâ€”a process where events triggered on a child element propagate up to its ancestors in the DOM.
Its usefulness: Efficiency: Only one event listener is added to the ul, reducing memory consumption and improving performance. Handling Dynamic Elements: If we later add more li elements dynamically, the event listener on ul will automatically handle clicks on the new items. Simpler Maintenance: Our code is easier to maintain because you're managing fewer event listeners and only dealing with the parent element.

How do you manipulate an elements attributes and styles using DOM

1. Manipulating Element Attributes: a. Getting an Attribute : element.getAttribute(''); b. Setting an Attribute : element.setAttribute('class', 'className'); c. Removing an Attribute: element.removeAttribute(''); d. If a class exist: let hasClass = element.hasAttribute(''); 2. Manipulating Element styles : a. Set a color: element.style.color = 'blue'; b. Get a style : let bgColor = element.style.backgroundColor; console.log(bgColor); c. Remove a style: element.style.color = ''; d. Add a class: element.classList.add('active');

When to use .value and when to use .innerText in DOM

When input and textarea text has to get we need to use .value but when we have to get text from tags like P, h1-h6 etc we need to use .innerText.
